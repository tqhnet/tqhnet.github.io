MotionBuilder2019

2020-09-23
软件

MotionBuilder是业界最为重要的3D角色动画软件之一。主要作用有两点，一是做动画，二是做动捕。它是可以增强任何现有制作生产线的补充软件包。

安装教程

1.双击【exe】安装文件，选择空余目录后解压，释放后文件夹大概1.6GB

2.选择【Install】

3.然后选择【I Accept】，点击【Next】

image

4.等待安装

image

5.安装完成后点击【Launch Now】

image

6.点击【enter a serial number】——【I agree】

image

7.点击【Activate】，进入激活。输入

序列号：666-69696969, 667-98989898, 400-45454545，066-66666666，

Key：727K1

单击【激活】之前，你有2个选择：

a.禁用网卡，拔出网线或阻止防火墙（这只是为了禁用在线检查）它会告诉你需要互联网连接

只需单击关闭，然后再次单击激活

b.点击激活，它将进行在线检查，只需点击关闭并再次单击激活。

选择选项a或b。

8.右键以管理员权限运行MotionBuilder 注册机，先点击【Patch】，提示“Successfully patched”并确定，把软件申请号复制到注册机的第一栏，然后点击【Generate】，生成激活码。

9.选中“【I have an activation code from autodesk】，把注册机第二栏生成的激活码，复制到下面的16个激活框里，然后点击【下一步】。

10.激活成功！点击【Finish】，这样MotionBuilder2019的安装就完成啦。
